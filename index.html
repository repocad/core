The internet of digital design

What is RepoCad?

RepoCad is a collaborative and modern design editor that uses the state of the art scripting language, RepoScript to write design. Together with our library of drawings, this gives you a novel way to collaborate on designs with whoever you want.

The editor

Instead of torturing users with repetitive strain injuries from pointing and clicking, RepoCad uses text to create visuals.

RepoScript turns code into design in an elegant way. Simplicity is key.

 Go to the editor

The library

RepoCad is a library of design drawings and tools that can be used by anyone.

We want to move the lonely designer into a global design network, where ideas are shared and built upon.

 Go to the library

Why RepoCad?

RepoCad is a result of many years frustration with current design tools and methods. Repo is short for repository, signifying the importance of online storage and accessibility. The title also contain a pun on "repossession": the re-claiming of tools used for creating designs.

Read more about why we created RepoCad on our documentation page.

Use cases

Storage boxes

A box for storing miscellaneous objects. This proof-of-concept uses parametricity to create a rectangular plate, which is repeated 5 times to form a box.

The box can be exported and sent to a laser cutter or similar manifacturing equipment.

  View in RepoCad


Parametric bridge

This bridge is constructed using simple parametric principles. The bridge support lines is drawn from the same lines code, but is drawn repeated at 6 points on the bridge.

  View in RepoCad


Patterns for Guzman (art)

This artwork by Ole Egholm Jackson, illustrates the more abstract uses of RepoCad. By toying with the editor for a few minutes, it is possible to create unique and complex geometries.

As with the above examples, we encourage you to enter the drawing and see what happens if the code is changed!
